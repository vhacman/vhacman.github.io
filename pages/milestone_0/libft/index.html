Libft

Custom C Library Implementation

About This Project

Libft is the first project at 42 School and serves as the foundation for all future C projects. This project involves recreating essential C standard library functions from scratch, providing a deep understanding of how these fundamental functions work under the hood.

The library includes implementations of string manipulation, memory management, character validation, list manipulation functions, and more. This custom library becomes a valuable tool that can be used and extended throughout the Common Core curriculum.

Skills Acquired

  • Deep understanding of C programming fundamentals
  • Memory allocation and pointer manipulation
  • String processing and manipulation techniques
  • Linked list data structure implementation
  • Code organization and library creation
  • Makefile creation and compilation process
  • Rigorous testing and debugging practices
