C 02

C Piscine - Strings

About This Project

C 02 focuses on strings in C, teaching students how to manipulate null-terminated character arrays. This project covers string copying, concatenation, comparison, and transformation operations, building essential skills for text processing in C.

Students implement their own versions of standard library string functions, gaining deep understanding of how strings work in C. The exercises emphasize pointer manipulation, array traversal, and character-by-character operations.

Skills Acquired

  • String fundamentals and null termination
  • String manipulation functions (strcpy, strcat, etc.)
  • Character array traversal and manipulation
  • String comparison and searching
  • Character classification and conversion
  • Working with ASCII values
  • Buffer handling and bounds checking
