C 03

C Piscine - String Functions

About This Project

C 03 continues the exploration of string manipulation with more advanced functions. Students recreate additional standard library functions like strcmp, strncmp, strcat, strncat, and strstr, deepening their understanding of string comparison and concatenation.

This project emphasizes the importance of edge cases, buffer sizes, and safe string operations. Students learn to handle partial string operations and understand the differences between bounded and unbounded string functions.

Skills Acquired

  • Advanced string comparison techniques
  • String concatenation and appending
  • Bounded vs unbounded string operations
  • Substring searching and pattern matching
  • Safe string handling practices
  • Edge case handling in string functions
  • Return value conventions in C
