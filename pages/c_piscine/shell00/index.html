Shell 00

C Piscine - Basic Shell Commands

About This Project

Shell 00 is the first project of the C Piscine, introducing students to the Unix command line and basic shell commands. This project covers essential operations like file creation, directory navigation, permissions management, and text manipulation using shell utilities.

Students learn to navigate the terminal environment, understand file permissions, use basic commands, and create simple shell scripts. It's the foundation for all command-line work throughout the 42 curriculum.

Skills Acquired

  • Basic Unix/Linux command line navigation
  • File and directory manipulation (ls, cd, mkdir, rm)
  • File permissions and chmod command
  • Text manipulation with cat, echo, and redirection
  • Understanding of the Unix file system
  • Basic shell scripting concepts
  • Working with hidden files and special characters
