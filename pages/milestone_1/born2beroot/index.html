Born2beroot

Amministrazione di Sistema e Virtualizzazione

System Administration & Virtualization

Informazioni sul Progetto

Born2beroot è un progetto di amministrazione di sistema che introduce i fondamenti della virtualizzazione e della configurazione dei server. Gli studenti devono configurare una macchina virtuale sicura seguendo linee guida rigorose, implementando varie politiche di sicurezza e servizi.

Il progetto copre attività essenziali di sysadmin tra cui partizionamento, configurazione delle politiche sudo, configurazione di SSH, implementazione di un firewall con UFW e creazione di script di monitoraggio. Fornisce esperienza pratica con distribuzioni Debian e Rocky Linux.

About This Project

Born2beroot is a system administration project that introduces the fundamentals of virtualization and server configuration. Students must set up a secure virtual machine following strict guidelines, implementing various security policies and services.

The project covers essential sysadmin tasks including partitioning, configuring sudo policies, setting up SSH, implementing a firewall with UFW, and creating monitoring scripts. It provides hands-on experience with both Debian and Rocky Linux distributions.

Competenze Acquisite

  • Configurazione e setup di macchine virtuali (VirtualBox)
  • Fondamenti di amministrazione di sistemi Linux
  • Configurazione di sudo e gestione dei privilegi
  • Setup del server SSH e hardening di sicurezza
  • Configurazione del firewall UFW e regole
  • Politiche delle password e gestione utenti
  • Scripting Bash per il monitoraggio del sistema
  • Comprensione di AppArmor/SELinux

Skills Acquired

  • Virtual machine setup and configuration (VirtualBox)
  • Linux system administration fundamentals
  • Sudo configuration and privilege management
  • SSH server setup and security hardening
  • UFW firewall configuration and rules
  • Password policies and user management
  • Bash scripting for system monitoring
  • Understanding of AppArmor/SELinux
